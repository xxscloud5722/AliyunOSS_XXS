# AliyunOSS_XXS
AliyunOSS_XXS 


### 简单说明
项目已经经过反复测试已经可以在阿里云官方DLL 基础上开发

1. 提供一个任务（上传任务、下载任务）管理类
2. 上传任务可以断点续传、委托函数回调
3. 下载任务可以断点续传、委托函数回执



### 使用方法
/build/*
项目编译好的DLL
